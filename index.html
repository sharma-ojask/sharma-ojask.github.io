Machine Learning & Data Science Projects

Global Forest Cover

An interactive data visualization and forecasting project that analyzes global forest cover trends throughout the 21st century. The tool enables users to explore historical patterns of deforestation and reforestation across regions and generates forward-looking forecasts to estimate future forest cover under observed trends.

Contributions: Led data preprocessing and feature preparation for large-scale environmental datasets, including cleaning, temporal aggregation, and regional alignment. Spearheaded the development of the forecasting pipeline, with primary ownership of the Prophet-based time series model used to generate future forest cover projections and uncertainty intervals.

WikiQA Span Extraction

A natural language processing project focused on extractive question answering using the WikiQA dataset. The system identifies and selects precise answer spans from Wikipedia passages in response to user questions, emphasizing alignment-based reasoning rather than generative text output.

Contributions: Assisted in the data processing pipeline, including tokenization, span alignment logic between questions and candidate passages, and handling edge cases in noisy annotations. Designed and implemented the evaluation methodology, ensuring accurate measurement of span-level performance using appropriate metrics and consistent alignment between predictions and ground truth.
