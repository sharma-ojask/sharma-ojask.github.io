Global Forest Cover Visualizer

An interactive visualizer that analyzes global forest cover trends throughout the 21st century. The tool enables users to explore historical patterns of deforestation and reforestation across regions and generates forward-looking forecasts to estimate future forest cover under observed trends.

  • Converted satellite imagery into pixel-level features for time series modeling
  • Led forecasting model development for global forest cover trends

WikiQA Span Extraction

A natural language processing project focused on extractive question answering using the WikiQA dataset. The system identifies precise answer spans from Wikipedia passages, emphasizing alignment-based reasoning rather than generative text output.

  • Implemented tokenization and span alignment for extractive QA
  • Handled noisy annotations and dataset edge cases
  • Designed span-level evaluation methodology

  • Contributed to dashboard development and narrative storytelling
